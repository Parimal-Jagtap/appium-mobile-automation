# appium-mobile-automation
Mobile test automation framework for Android and iOS using Appium (Java), UIAutomator2, XCUITest, and TestNG. Covers fintech mobile workflows including transactions, wallet, and payments.

# 📱 Appium Mobile Automation Framework

> Mobile test automation for **Android & iOS** banking/fintech applications  
> Built with **Appium + Java + TestNG** | UIAutomator2 | XCUITest | CI/CD via **GitHub Actions**
